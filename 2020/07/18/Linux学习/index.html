Linux备忘录

记录linux的文件目录和常见命令等

Linux认知

一款免费的，开源的操作系统，继承自unix系统，是GNU人机交互理念的操作系统层，只是一个内核，不同的厂商提供了不同的包装，如centOS，Redhat，Ubuntu，Suse，红旗Linux等。

Linux安装

先下载一款软件VM，通过VM在物理机上划分出一块空间作为虚拟机，在这块空间上安装Linux系统。注意要在BIOS中开启虚拟化设备支持（Intel Virtual Technology）。在指定虚拟机参数时建议根据物理机的性能来指定，虚拟机的网络连接模式分为三种，分别是：

  1. 桥连。在这种网络连接模式下，虚拟机可以和其他系统通信，但是会占据当前局域网的一个ip地址，所以可能会导致ip冲突。
  2. NAT网络地址转换方式。虚拟机可以访问外网，使用非本局域网的ip地址，不会造成ip冲突。
  3. 主机模式。虚拟机作为一个独立主机，不能访问外网。

Linux目录结构

Linux是一个树状结构，最上层是根目录，各个目录存放的内容预先都规划好了，在Linux中，一切皆是文件。

根目录下主要分为这些目录：

  • /bin ：Binary的缩写，存放最常用的命令。

  • /sbin：管理员使用的系统管理程序。

  • /home：存放普通用户的主目录，每个用户都有自己的目录，一般以用户账号命名。

  • /root：系统管理员的用户主目录。

  • /lib：系统开机所需要的最基本的动态链接数据库（相当于Windows中的DLL）。

  • /lost+found：一般为空，存放系统非法关机后的某些文件。

  • /etc：所有系统管理所需要的配置文件和子目录。

  • /usr：用户的应用程序和文件目录。

  • /boot：启动Linux的核心文件

  • /tmp：临时文件。

  • /dev：硬件资源映射的文件（太强了）

  • /media：识别的设备如u盘等挂载进此文件目录。

  • /opt：安装软件所在的目录（未安装的压缩包）。

  • /usr/local：另一个给主机额外安装软件的安装目录（安装后的软件）。

  • /var：日志文件目录。

  • /selinux：安全子系统，当系统被攻击时触发。

Vim编辑器

vim编辑器有三种模式

  • 正常模式：默认模式，可以使用快捷键。
  • 插入模式：可以在正常模式下按下i/I，o/O，a/A，r/R等进入，一般使用i
  • 命令行模式：提供相关指令，完成读取、存盘、替换、离开vim、显示行号等动作。

模式转换：

  • vim xxx 进入一般模式。
  • i 进入插入模式。
  • ESC返回一般模式。
  • ：或/ 进入命令模式。wq：保存并退出。q：退出。q！：强制退出。

vim快捷键：

  • nyy：拷贝当前n行，粘贴（P）。
  • ndd：删除当前n行。
  • /string：查找关键词，输入n查找下一个。
  • set nu / set nonu 设置文件行号。
  • 插入模式下：G 到最末尾，gg：到首行。
  • u：撤销操作。（正常模式下）
  • set nu n shift -g ： 光标移动到第n行。

Linux常用命令

关机重启指令

  • shutdown：可以接多个选项。

    • -h now 立即关机。

    • -h x x分钟后关机。

    • -r now 立即重启。

  • halt：相当于shutdown -h now 立即关机。

  • reboot：相当于shutdown -r now 立即重启。

  • sync：将内存数据同步到磁盘，建议关机前执行这个指令。

用户登录与注销
  • logout：退出当前用户。（注：图形界面无效。）
用户管理
  • useradd 【可选项】 用户名：添加用户，其中【可选项】分为
    • 【-d】：指定用户目录
    • 【-g】指定用户组。
  • passwd 用户名：指定/修改用户密码。
  • userdel 【可选项】 用户名：删除用户。【可选项】：
    • 【-r】：删除用户并不保留家目录。
  • id 用户名：查看用户。查看结果分为
    • uid：用户id。
    • gid：用户组id。
    • 组：用户组信息。
  • su -用户名：切换用户。高权限用户到底权限用户无需输入密码。
    • exit：回退操作，返回原用户。
    • whoami：查看自身用户。
用户组管理
  • groupadd 组名：增加用户组。
  • groupdel 组名：删除用户组。
  • usermod -g 用户组 用户名：修改用户所在的组。
  • useradd -g 用户组 用户名：指定用户所在的组。

用户信息存放在：/etc/passwd中，每行数据的显示格式如下：

​ 用户名：口令：用户标识号：组标识号：注释性描述：主目录：shell相关

组信息存放在：/etc/group中，每行数据显示格式如下：

​ 组名：口令：组标识号：组内用户列表

口令相关信息存放在：/etc/shadow中，没行数据显示格式如下：

​ 登录名：加密口令：最后修改时间：最小时间间隔：最大时间间隔：警告时间：不活动时间：失效时 间：标志

运行级别
  • init 【0123456】：指定运行级别。

linux系统一共七个运行级别：

  • 0：关机

  • 1：单用户

  • 2：多用户无网络

  • 3：多用户有网络

  • 4：保留级别

  • 5：图形界面

  • 6：重启

如果忘记了root密码，可以在开机时指定为单用户模式进入root，重新指定root密码。

帮助指令
  • man 【命令或配置文件】：显示命令或配置文件的相关文档。
  • help 【命令】：显示命令的使用教程。
文件目录类
  • pwd：显示当前目录的绝对路径
  • ls 【可选项】：显示文件或目录的信息，【可选项】：
    • 【-a】：包括隐藏文件。
    • 【-l】：列表排列。
  • cd 目录：切换到想去的目录。
    • cd ~ ： 回到该用户的家目录。
    • cd .. ：回到上一级目录。
  • mkdir 【可选项】 目录名：创建目录。【可选项】：
    • -p：创建多级目录。
  • rmdir 目录名：删除目录。（目录中有文件则无法删除）
  • rm 【可选项】：移除文件或目录。【可选项】：
    • -f：删除时不询问。
    • -r：递归删除文件夹下所有目录。
  • touch 文件名：创建一个空文件。
  • cp 【选项】 source dest：拷贝指定文件到指定目录，【可选项】：
    • -r：递归复制文件夹。
    • \cp：强制覆盖不提醒。
  • mv 旧文件 新文件/路径：重命名文件/移动文件。
  • cat 【可选项】 文件名：查看文件内容（只读），【可选项】：
    • -n：显示行号。
    • | more：分页显示。（放在末尾）
  • more 文件名：查看文件内容（文本过滤器）并分页显示，显示时可以使用以下按键：
    • space：下一页。
    • enter：下一行。
    • q：离开more，不显示文件内容。
    • ctrl F：下滚一屏。
    • ctrl B：上滚一屏。
  • less 文件名：类似more，但是缓存显示，用于查看大型的日志文件。
  • > : 输出重定向，将原文件内容覆盖。
  • >>：追加。将内容追加到原文件尾部。
  • echo：输出内容到控制台。
  • head 【可选部分】 文件名 ：显示文件的开头部分（默认10行），【可选部分】：
    • -n x：显示开头x行。
  • tail 【可选部分】 文件名：显示文件末尾部分（默认10行），【可选部分】：
    • -n x，显示末尾n行。
    • -f：实时追踪文档的所有更新。
  • ln -s 源目录 软链接文件：软链接指令。此时pwd查看的仍是软链接目录。
  • history：查看已经执行的历史命令。
时间日期类
  • date：显示当前时间。
    • +%Y：显示年份。
    • +%m：显示月份。
    • +%D：显示日子。
    • +%H：显示小时。
    • +%M：显示分钟。
    • +%S：显示秒钟。
  • cal：查看日历情况。
搜索查找类
  • find 【搜索范围】 【可选项】：在某个范围内查找，【可选项】：
    • -name<文件名>：根据名称寻找。
    • -user<用户名>：根据用户寻找。
    • -size<文件大小>：根据文件大小寻找。这里+代表大于，-代表小于。
  • locate 文件名：利用实现建立的locate数据库实现快速查找。但是查找直线需要先建立数据库。
  • grep 【可选项】 查找内容 | 源文件：利用管道符号|将前一个命令的处理结果输出传递给后面的命令处理。
    • -n：显示行号。
    • -i：忽视大小写。
压缩与解压缩
  • gzip：压缩文件。
  • gunzip：解压缩。
  • zip -r ：递归压缩文件。
  • unzip -d ： 将文件解压到指定的位置。
  • tar 【可选项】：打包指令。【可选项】：
    • -c：产生tar打包文件。
    • -v：显示详细信息。
    • -f：指定打包后的文件名。
    • -z：打包的同时压缩。
    • -x：解包tar文件。

组管理

Linux每个用户属于一个组。不能独立于组外。Linux中每个文件都有所有者，所在组，其他组的概念。

文件操作：
  • 修改文件所有者： chown 新用户 文件名
  • 修改文件所在组： chgrp 新组名 文件名
用户操作：
  • 修改用户所在组： usermod -g 新组名 用户名

权限管理

给一个文件或者目录指定三个权限，分别是该文件或目录所有者拥有的权限、所有者所在组拥有的权限、其他组拥有的权限。

文件类型：
  • - ：普通文件
  • d : 目录
  • l ： 链接文件
  • c ：字符设备（键盘、鼠标等）
  • b ： 块文件（硬盘）
权限说明：
  • r ： 文件的r代表可读，目录的r代表可查看。
  • w ： 文件的w代表可写，目录的w代表可修改（删除修改目录中的文件等）。
  • x ：文件中的x代表可执行，目录中的x代表可进入。
修改权限：
方式一：使用 + 、- 、 = 指定权限
  • chmod u/g/o/a =/+/- r/w/x 文件/目录 （u：所有者 g：所在组 o：其他组 a：所有人）
方式二：通过数字指定：
  • chmod 751 文件/目录 （r：4 w：2 x：1）

任务调度

某些场景下我们希望在一个固定的时间执行一个固定的操作，如每天凌晨四点备份数据库，可以使用任务调度方式。

语法：

crontab 【选项】，其中【选项】可选值为：

  • -e ：编辑crontab定时任务
  • -l ： 查询crontab任务
  • -r ： 删除当前用户的所有crontab任务
步骤：
  1. 编辑任务脚本
  2. 给予该脚本可执行权限
  3. 使用 crontab -e 脚本的方式指定任务调度
  4. 保存退出使任务调度生效
参数说明:
  • 第一个*：一小时的第几分钟，可选值1-59
  • 第二个*：一天的第几个小时，可选值1-23
  • 第三个*：一个月的第几天，可选值1-31
  • 第四个*：一年的第几个月，可选值1-12
  • 第五个*：一周中的星期几，可选值0-7（0和7都代表周日）
