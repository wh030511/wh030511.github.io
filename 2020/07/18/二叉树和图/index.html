这里对ArrayDeque做一个简要介绍：

  • ArrayDeque是Deque的一个实现类，Deque接口继承了Queue接口，可以用来作为栈和队列使用。优点在于比Stack作为栈和LinkedList作为队列时效率更高。

  • 不能存放null值

  • 其底层结构是使用数组和两个指针实现的，其中tail指针永远指向最后一个元素的下一个位置。数组长度永远会保证是2的幂次，如果初始化指定长度时就是2的幂次，那么最终幂次会+1

  • 其扩容机制会扩容成当前容量的两倍。这个数据结构不是线程安全的。

    作为栈：

    push（）：调用addFirst（）方法

    pop（）：调用removeFirst（）方法

    remove和poll的区别还是和LinkedList一样的，即remove访问到null时会抛出异常，而poll会返回null

    作为队列：

    add（）/ offer（）：调用相关的xxxLast（）方法

    remove（）/ poll（）：调用相关的xxxFirst（）方法

    element（）/ peek（）：返回头部元素

    null值时，element抛异常，peek返回null

    满队列添加新元素时，add抛异常，offer返回false

4. 二叉树的最近公共祖先

给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。

百度百科中最近公共祖先的定义为：“对于有根树 T 的两个结点 p、q，最近公共祖先表示为一个结点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”

思路：对于p、q两个节点，找到从根节点到他们的路径，放到一个队列中，然后从头遍历这两个队列，最后一个相等的元素即为他们的最近公共祖先。
